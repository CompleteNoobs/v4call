v4call — Rate Editor v2

Full rate schedule editor · Posts to Hive blockchain via Keychain
IDENTITY
BLOCKLIST
TOKEN RATES
Accept Hive-Engine tokens (PIZZA, HIVE, your own token…) in addition to HBD. Add one section per token. Enter 0 for FREE. Token holders can optionally bypass your blocklist.
RATE LISTS
Named lists of users get custom rates. Lists are checked in order — first match wins. Anyone not matched uses the default rates. Enter 0 in any field to make it FREE.
OUTPUT — V4CALL-RATES-V2
Posting requires the Hive Keychain browser extension. Post title: v4call-rates · Tag: v4call. The app always reads your most recent post with that title.
RATE REFERENCE
RING — fee to make your device ring
CONNECT — fee when you accept
RATE /HR — per-hour fee, prorated per minute
TEXT — flat fee per message
TEXT SESSION /HR — ongoing text session fee
MIN-DEPOSIT — minutes of call time pre-paid before connecting
0 = FREE — any field set to 0 outputs FREE
HBD rates → HBD (≈ USD)
Token rates → token amounts
Times → your local timezone
Duration → prorated per minute
